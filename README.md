## Lab 7
### Swiper.js
Выбрал эту библиотеку, потому-что существует огромное количество визуального контента который я бы хотел отобразить на своей странице.
Для удобного размещения изображений в "карусели" и существует эта библиотека.

Use-case сценарий:

Отображения нескольких заглавных изображений на index странице, с возможностью переключения между ними
с помощью кнопок-стрелок. Это позволит сэкономить место на странице, сохранив изначальные размеры изображений.
